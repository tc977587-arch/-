汽车云留言板
一个基于 HTML5 Canvas 和 PHP 开发的极简匿名手写留言系统。其核心特性在于交互设计：用户发送留言后，笔迹将脱离画板容器，垂直向上缓慢滑出网页边界，模拟留言升空的视觉效果。

项目特性
手写交互：基于原生 Canvas API 捕捉笔迹，完美适配移动端触控与 PC 端鼠标操作。

影子克隆动画：采用 JS 动态创建克隆元素并注入 Body 层级，解决了笔迹在跨越父级容器边界时的显示剪裁问题。

慢速升空特效：通过 CSS3 Transition 实现 8 秒匀速垂直位移，提供沉浸式的发送反馈。

轻量化架构：无需配置数据库。采用物理文件存储方案，利用 PHP 处理图像流并保存为 PNG 格式。

文件结构
index.html：前端画板界面及笔迹平移逻辑。

save_msg.php：后端处理程序，负责 Base64 编码解码及文件持久化。

admin.php：管理员后台，用于按时间降序查看所有存储的留言图片。

/messages：留言图片存储目录（需手动创建并配置权限）。

环境要求
操作系统：Linux / Windows / macOS

服务端：Nginx 或 Apache

语言环境：PHP 7.4 及以上版本（推荐 PHP 8.2）

部署流程
创建站点：在 Web 环境（如宝塔面板）中新建 PHP 站点，并将解析域名指向该站点。

上传源码：将项目中的核心文件上传至网站根目录。

配置目录权限：

在根目录下手动创建 messages 文件夹。

关键步骤：将该文件夹的写入权限设为 755，并将所有者变更为 Web 服务用户（如 www）。

访问测试：访问前端域名进行书写测试，并检查 admin.php 是否能正常渲染生成的图片。

安全建议
由于 admin.php 默认不带鉴权逻辑，建议通过 Web 服务器配置（如 Nginx 的 auth_basic）对后台页面添加访问限制，以保护用户隐私。
